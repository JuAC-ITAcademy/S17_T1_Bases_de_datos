# S17_T1_Bases_de_datos
Tarea 1 del Sprint 17. Bases de datos relacionales.

En esta tarea voy a poner en práctica los conceptos **ENTIDAD** - **RELACIÓN** y voy a diseñar desde cero una base de datos que registre las transacciones de una red de tiendas.

Además daré los primeros pasos con el workbench de mySQL y crearé el primer diagrama relacional de bases de datos.
